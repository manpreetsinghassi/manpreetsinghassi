Soundcloud

Overview

SoundCloud is an online audio platform that allows users to upload, promote, and share their music, podcasts, and other audio recordings with a global audience. The platform features a vast collection of tracks, ranging from original compositions by independent artists to remixes and cover versions of popular songs. This case study focused on the app's UI and some improvement to the UX.

Role

UI/UX designer

Tools Used

Figma

Problem Analysis

The existing SoundCloud UI suffered from issues with consistency, clutter, and user experience. The interface did not make optimal use of available screen space, resulting in a suboptimal user experience."

My Solution

To address these problems, the UI was redesigned to improve consistency, simplify the interface, and make better use of space. Specifically, background colors were added to distinguish content types, and certain features were made collapsible.
